### How to use the terminal
#### *adding exercises*
    add dips 20kg x5
    add dips 20kg x5x5x5
    add dips bw 3x10
#### *adding cardio*
    add zone 2 30min cyclette
#### *adding focus sessions*
    add focus 30min
#### *setting a goal or updating stats*
    set goal dips 35kg x5
    set bw 75kg
